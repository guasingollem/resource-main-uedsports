# resource-main-uedsports

## 项目简介

`resource-main-uedsports` 是一个用于归档和托管多个独立 HTML 页面的仓库。本仓库不针对任何特定网站或域名，仅作为通用 HTML 资源文件的存储与发布中心。

## 目录结构

```
.
├── index.html          # 仓库首页 / 导航页
├── pages/              # 存放各独立 HTML 页面
│   └── ...
├── assets/             # 公共资源（CSS、JS、图片等）
│   └── ...
└── README.md           # 本文件
```

- `pages/` 目录下存放各个独立的 HTML 页面文件，文件名应具有描述性，便于识别。
- `assets/` 目录用于存放页面所依赖的公共样式、脚本及媒体资源。

## 页面归档说明

本仓库中的 HTML 页面均为独立归档，彼此之间无强制依赖关系。每个页面均可通过直接访问其路径来查看，无需额外配置服务器环境。

如果您需要添加新的页面，请遵循以下原则：

- 将 HTML 文件放入 `pages/` 目录。
- 如有公共资源依赖，请放入 `assets/` 目录，并在 HTML 中通过相对路径引用。
- 避免在页面内引用外部不可控资源，以保证归档的长期可用性。

## 维护说明

本仓库由维护者不定期更新。欢迎通过 Issue 提交建议或报告问题，也接受符合归档目标的 Pull Request。

提交新页面时，请确保：

- 文件无恶意代码或外部追踪脚本。
- 页面内容清晰、结构完整。
- 不包含针对特定域名或商业推广的链接。
